cube
====

Spin a cube. Stamp it with your word. Enjoy the show.

Works on Linux, macOS and Windows.

# Install it
```bash
cargo install cube
```

Run it
------
```bash
cube "your label"
```


- Label: max 256 chars, defaults to `cube`.
- Esc to quit.
